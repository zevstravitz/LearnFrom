Natural - the set of positive counting numbers, i.e 1,2,3,...

Integers - the set of natural numbers, their negative counterparts , and 0. These are often thought of as the values on a number line

Rational - Any number which can be expressed as a fraction, i.e 21/4, -â…š, 3, and 123/4. Any integer is also a rational number

Irrational - Any number which cannot be expressed as fraction, in decimal form they neither terminate or repeat

Real - a value that can represent a point on a line, it can be either positive or negative, both rational and irrational numbers are included in this set

Imaginary - numbers that can be written in the form bi, where b is a real number and i is the number sqr(-1)

Complex - numbers that can be represented in the form a+bi, where a is a real number and b is a imaginary number
